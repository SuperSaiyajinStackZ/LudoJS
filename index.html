This is a Ludo3DS port to JS, HTML and CSS.
Version: v0.2.0

Features including:

* Fully functional gameplay.
* Ability to load an already started game from LudoJS or Ludo3DS.
* Ability to save the current game to a file for later re-use. Can also be used for Ludo3DS.
* An optional computer opponent.
